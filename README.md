# **서울시_부동산 데이터 분석 프로젝트**
2018~2024_서울시 부동산 실거래가 데이터분석_개인 작업물 모음입니다. 

작업물은 `개인작업물` 폴더 내부를 확인해주세요.

## 프로젝트 요약
- 목표: 2018~2024년 서울시 부동산 실거래가 데이터를 분석해 거주 목적 매수 고객의 페르소나별 맞춤 매물 추천 서비스 제공
- 기간(팀 규모): 2026.01.23 ~ 2026.02.02 (5인)
- 개요
  - 서울시 실거래가 데이터(약 81만 건)를 MySQL로 정제 및 가공해 분석용 테이블을 설계하고, 자치구별 부동산 가격 변동 패턴을 파악함
  - 신혼부부/청년 1인가구/은퇴를 앞둔 2주택 보유자로 구성된 페르소나의 요구사항에 맞는 부동산 매물을 각각 추천함
