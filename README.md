## ⚠️ **GitHub 자체 주피터 노트북(.ipynb) 렌더링 에러 안내**

현재 GitHub 전체 시스템의 일시적인 주피터 노트북 렌더링 서버 결함(`nbformat` 호환성 오류)으로 인해 일부 `.ipynb` 파일이 정상적으로 열리지 않는 현상이 발생하고 있습니다.

본 프로젝트의 핵심 역량인 **데이터 전처리 알고리즘 및 분석 파이프라인**은 GitHub 버그와 무관하게 즉시 확인할 수 있는 가독성이 높은 마크다운 문서(`.md`), 파이썬 스크립트(`.py`), SQL 쿼리(`.sql`) 형태로 업데이트해 두었습니다. 부디 너른 양해 부탁드립니다.
