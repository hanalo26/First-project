# 서울시 부동산 매매 실거래가 분석 (MySQL)

페르소나 기반 서울시 아파트·연립다세대·오피스텔 매매 실거래가 분석 프로젝트입니다.  
공공데이터를 정제·가공하여 페르소나별 맞춤 매물 리스트를 추출했습니다.

---

## 파일 구조

```
 real-estate-sql/
 ├── 01_preprocessing.sql   # 원본 데이터 정제 및 파생 변수 생성
 ├── 02_analysis.sql        # 가격 안정성 지표(CV) 계산 및 분석 테이블 생성
 ├── 03_persona1.sql        # 페르소나 1 : 신혼부부 첫 집 마련형
 ├── 04_persona2.sql        # 페르소나 2 : 청년 1인가구 첫 매매
 └── 05_persona3.sql        # 페르소나 3 : 2주택 보유자 실거주 이전형
```

---

## 주요 테이블

| 테이블명 | 생성 파일 | 설명 |
|---|---|---|
| `prop_inline` | 01_preprocessing.sql | 전처리 완료된 메인 매물 테이블 |
| `cal_for_cv` | 02_analysis.sql | 구·동·건물용도·연도별 가격 변동계수(CV) 테이블 |
| `persona1` | 03_persona1.sql | 페르소나 1 조건 필터링 매물 테이블 |
| `persona2` | 04_persona2.sql | 페르소나 2 조건 필터링 매물 테이블 |
| `persona3` | 05_persona3.sql | 페르소나 3 조건 필터링 매물 테이블 |

### 주요 컬럼

| 컬럼명 | 설명 |
|---|---|
| `RCPT_YR` | 부동산 매물 접수 연도 |
| `CGG_NM` | 자치구명 |
| `STDG_NM` | 법정동명 |
| `BLDG_NM` | 건물명 |
| `CTRT_MONTH` | 계약한 연월 (YYYY-MM) |
| `CTRT_QUARTER` | 계약한 시기 (YYYY-QN) |
| `THING_AMT` | 매매가 (만원) |
| `FLR` | 층 |
| `BLDG_USG` | 건물용도 (아파트 / 연립다세대 / 오피스텔 / 단독다가구) |
| `BD_AGE` | 건물연식 (년) |
| `K_SQ` | 면적 (평) |
| `PRICE_PER_KSQ` | 평당가 (만원/평) |
| `is_PM` | 프리미엄 매물 여부 (1 = 프리미엄, 0 = 일반) |
| `CV_of_price` | 가격 변동계수 — 낮을수록 가격 안정적 |

## 페르소나 소개

### 페르소나 1 — 신혼부부 첫 집 마련형 (아파트 선호)

> 차우(32, IT 개발자 · 여의도) × 강혜리(30, 마케팅 직장인 · 강남)  
> 현재 마포구 거주 | 예산 8억 이하 | 소형 SUV 1대 보유

| 구분 | 조건 |
|---|---|
| 건물 용도 | 아파트 |
| 희망 지역 | 성동구 · 광진구 (2호선 역세권) |
| 매매가 | 8억 이하 (`THING_AMT <= 80000`) |
| 평수 | 20 ~ 30평 (`K_SQ BETWEEN 20 AND 30`) |
| 건물 연식 | 15년 이하 (`BD_AGE <= 15`) |
| 프리미엄 매물 | 제외 (`is_PM = 0`) |

---

### 페르소나 2 — 청년 1인가구 첫 매매 (전세 탈출)

> 천준석(36, 데이터 분석가 · 판교)  
> 현재 관악구 거주 | 예산 4억 이하 | 강아지와 동거 중

| 구분 | 조건 |
|---|---|
| 건물 용도 | 연립다세대, 오피스텔 |
| 희망 지역 | 관악구 (신림동 · 남현동 · 봉천동) |
| 매매가 | 4억 이하 (`THING_AMT <= 40000`) |
| 건물 연식 | 15년 이하 (`BD_AGE <= 15`) |
| 프리미엄 매물 | 제외 (`is_PM = 0`) |

---

### 페르소나 3 — 2주택 보유자 (투자성 오피스텔 보유 + 실거주 이전)

> 설무아(58, 중견기업 임원) × 김혜지(56, 공공기관 행정직)  
> 현재 송파구 거주 | 예산 20억 이하 | 대형 SUV 1대 보유

| 구분 | 조건 |
|---|---|
| 건물 용도 | 아파트 |
| 희망 지역 | 강동구 |
| 매매가 | 20억 이하 (`THING_AMT <= 200000`) |
| 평수 | 20 ~ 40평 (`K_SQ BETWEEN 20 AND 40`) |
| 건물 연식 | 15년 이하 (`BD_AGE <= 15`) |
| 프리미엄 매물 | 제외 (`is_PM = 0`) |

## 매물 리스트 출력 기준

각 페르소나 파일의 매물 리스트 조회 쿼리는 아래 기준으로 정렬·필터합니다.

- **건물명 중복 제거** : 동일 건물이면 가장 최근 접수연도 데이터만 출력
- **정렬 우선순위** : 최근 접수연도 → CV 낮은 순 (가격 안정적) → 매매가 높은 순
- **출력 건수** : 상위 30건 (`LIMIT 30`)

## 실행 방법

```sql
-- 1. 전처리 (원본 → prop_inline)
SOURCE 01_preprocessing.sql;

-- 2. 분석 테이블 생성 (CV 계산)
SOURCE 02_analysis.sql;

-- 3. 페르소나별 매물 테이블 + 리스트 조회
SOURCE 03_persona1.sql;
SOURCE 04_persona2.sql;
SOURCE 05_persona3.sql;
```

> **환경** : MySQL 8.0 이상 권장
